# Game Design

## 한 줄 콘셉트

제한된 잉크로 공격형, 방어형 말을 직접 설계한 뒤 실제 형태와 물리 특성을
이용해 상대 말을 밀어내는 픽셀 알까기 게임.

## MVP 플레이 흐름

1. 메인 화면
2. 로컬 대전 선택
3. 1P 말 제작
4. 2P 말 제작
5. 1P 말 배치
6. 2P 말 배치
7. 선공 결정
8. 턴제 대전
9. 승패 판정
10. 재대결 또는 메인 화면

## 핵심 규칙

- 플레이어는 각각 3개의 말을 만든다.
- 플레이어는 총 60칸의 잉크를 사용한다.
- 말 하나는 최소 10칸, 최대 32칸을 사용한다.
- 세 말을 모두 만든 뒤 총 60칸을 모두 사용해야 한다.
- 한 턴에는 자신의 말 하나만 발사할 수 있다.
- 아군끼리도 충돌하며, 아군 말도 판 밖으로 나가면 탈락한다.
- 상대 말 3개를 모두 판 밖으로 밀어내면 승리한다.
- 양쪽 마지막 말이 동시에 탈락하면 무승부다.

## 말 제작

말은 8x8 픽셀 그리드에서 한 번의 연속된 드래그로 만든다. 이미 칠한 픽셀을
다시 지나가는 것은 허용하며, 추가 잉크를 소비하지 않는다.

상하좌우 연결만 유효하다. 대각선만 맞닿은 픽셀은 연결로 인정하지 않는다.
빠른 드래그로 입력 이벤트 사이 간격이 커져도 중간 픽셀이 누락되면 안 된다.

## 전략으로 인정되는 형태

- ㄷ 내부에 작은 아군 말을 숨긴다.
- ㄷ 또는 ㅁ 내부 빈 공간에 다른 말이 들어간다.
- 긴 막대를 끝에서 쳐서 회전 공격을 만든다.
- ㄹ 또는 비대칭 형태로 예측하기 어려운 회전을 만든다.
- 구멍과 오목한 부분을 공격 및 방어에 활용한다.

이런 행동은 버그가 아니라 게임의 핵심 전략이다.

## MVP에서 제외하는 기능

- 온라인 대전
- 초대 코드
- 계정, 로그인, 데이터베이스
- 전적, 랭킹, 공개 매칭
- AI 상대, 채팅, 관전, 리플레이
- 스킨, 아이템, 특수 능력
- 서버, WebSocket
