# AGENTS.md

이 프로젝트에서 Codex와 다른 AI 코딩 에이전트는 아래 규칙을 지킨다.

## 절대 지켜야 할 게임 규칙

- 게임 규칙을 임의로 변경하지 않는다.
- ㄷ, ㄹ, ㅁ, 구멍, 오목한 부분을 물리적으로 유지한다.
- 오목한 부분을 자동으로 채우거나 Convex Hull로 감싸지 않는다.
- 사용자가 그린 모양을 원, 네모, 단순 외곽선으로 바꾸지 않는다.
- 화면에 보이는 픽셀과 실제 Fixture 위치가 어긋나지 않게 한다.
- 말 하나는 여러 픽셀 Fixture를 가진 하나의 Body로 만든다.
- 말의 빈 공간은 실제 빈 공간으로 유지한다.

## 코드 구조 규칙

- 게임 규칙, 물리 시뮬레이션, Canvas 렌더링, 사용자 입력을 분리한다.
- 물리 상수는 `src/config/physicsConfig.js`에 모은다.
- 게임 밸런스와 잉크 규칙은 `src/config/gameConfig.js`에 모은다.
- 물리 계산과 Canvas 렌더링을 분리한다.
- 직렬화가 필요한 원본 데이터에는 Planck Body 같은 런타임 객체를 넣지 않는다.
- 복잡한 좌표 변환, 무게중심, 충격량 계산에는 한국어 주석을 남긴다.
- 자명한 변수 대입에는 불필요한 주석을 남기지 않는다.

## MVP 범위

- 온라인 기능을 현재 단계에서 구현하지 않는다.
- WebSocket, 초대 코드, 방 생성, 서버, 데이터베이스 코드를 추가하지 않는다.
- 새로운 기능을 추가하기 전에 MVP 범위를 확인한다.
- 기획에 없는 부가 기능을 임의로 추가하지 않는다.

## 작업 습관

- 의미 있는 변경 후 `CHANGELOG.md`를 갱신한다.
- 작업 후 `npm run test`와 `npm run build`를 실행한다.
- 임시 코드나 사용하지 않는 파일을 방치하지 않는다.
- 비밀키나 환경변수가 필요한 코드는 현재 추가하지 않는다.
