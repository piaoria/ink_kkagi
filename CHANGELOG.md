# Changelog

## 0.1.0

### Added

- Vite 기반 프로젝트 설정 추가
- GitHub Pages 배포 workflow 추가
- Galmuri 기반 픽셀 도트 폰트 적용
- 네모 도트 컨셉의 메인 화면과 픽셀 로고 추가
- 말 3개 제작 흐름, 지우개 모드, 잉크 검증 로직 추가
- 1P/2P 말 배치 화면, 회전, 진영/겹침/보드 경계 검증 추가
- 빠른 시작용 기본 말 프리셋과 자동 배치 로직 추가
- 배치 화면의 자동 배치 버튼 추가
- 배치 완료 후 진입하는 경기 화면과 현재 턴/말 선택 UI 추가
- 선택한 말을 뒤로 끌어 조준하는 발사 준비 UI 추가
- Planck 기반 말 발사 시뮬레이션과 낙장 제거 처리 추가
- 발사 후 턴 전환과 낙장 결과 표시 추가
- 승패/무승부 판정 로직과 경기 종료 화면 추가
- 빠른 재시작 흐름 추가
- PWA manifest, 앱 아이콘, service worker 추가
- 모바일 safe-area, 터치 영역, 스크롤 튐 방지 보정 추가
- 선택한 말 중심에서 조준선이 시작되도록 조준 UI 보정
- 물리 결과를 격자에 맞출 때 말의 형태와 잉크 수가 보존되도록 보정
- 발사 직후 물리 결과를 짧은 프레임 애니메이션으로 재생하도록 개선
- 제작/배치/조준/물리/승패 규칙 단위 테스트 추가

### Changed

- 로컬 대전 버튼을 실제 제작 화면으로 이동하도록 변경
- 메인 화면에 `빠른 시작`과 `직접 제작` 진입점을 분리
- 제작/배치/경기 화면의 깨진 한글 문구를 정상 문구로 정리
- 총 잉크를 36칸, 말 크기를 7-18칸으로 조정
- 배치 보드를 14x20, 각 진영 10줄로 확대
- Vite와 Vitest를 업데이트해 `esbuild` 개발 서버 취약점 알림 제거
- CI clean install을 위해 `@emnapi` peer dependency 명시

### Known Issues

- 물리 결과는 아직 고정 step 후 격자에 재투영하는 MVP 방식입니다.
- 실제 충돌 손맛, 회전 감각, 모바일 조준선 위치는 추가 조정이 필요합니다.
